3. DISTRIBUTION
-------------------------------------------------------------------------------
Catalog of applications, tools, modules and files included in the distribution:

  * Notepad++

    - 7.9.2 32-bit (last Windows XP compatible version)

  * Notepad++ patches for Windows XP

    - libcurl 7.74.0 32-bit

  * Notepad++ PythonScript plugin

    - 1.5.4.0 32-bit

      - Python 2.7.18 32-bit

  * Notepad++ PythonScript plugin Python updates

    - Python 2.7.18 dlls

      - _ctypes.pyd

    - Python 2.7.18 modules

      - psutil 5.9.5

  * WinMerge

    - 2.16.8 32-bit

  * Visual C++ 2008 Redistributables

    https://www.catalog.update.microsoft.com/Search.aspx?q=kb2538243
    Dependency for the Python 2.7.x DLL linked with Notepad++ PythonScript
    plugin.

    - 9.0.30729.5677

See the installation log in autogenerated log directory - `.log`, for details
on executables and setups in the catalog.
